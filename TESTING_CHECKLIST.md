## 🔧 后端 API 测试

### 测试用例 10：API 连接

在浏览器 F12 → Network 标签中查看：

```
当搜索 "周杰伦" 时，应看到：
```

- [ ] **Request URL** → `/api/search?action=search&keyword=%E5%91%A8%E6%9D%B0%E4%BC%A6`
- [ ] **Status** → 200 (成功)
- [ ] **Response Time** → < 3 秒
- [ ] **Response** → 包含歌曲数组

**示例 Response：**
```json
{
  "success": true,
  "data": [
    {
      "id": "1",
      "title": "晴天",
      "artist": "周杰伦",
      "audioUrl": "/api/proxy?id=1&source=netease"
    }
  ]
}
```

### 测试用例 11：音频代理

在浏览器 F12 → Network 中查看音频请求：

- [ ] **请求 URL** → `/api/proxy?id=xxx&source=netease`

- [ ] **Content-Type** → audio/mpeg
